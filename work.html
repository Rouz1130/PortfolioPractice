BAND TRACKER

ABOUT THE PROJECT

A software appliction to track both bands and venues to view where each band played concerts at. As a user you are allowed to create bands and assign them to a particular venue. A venue can host many bands, and a band can play at any venue that has been created. A user can also view a single venue to list out all of the bands that have played at that venue and vice versa for a venue to list out all the bands that played there.


LANGUAGES AND TOOLS

SQL database server integration and C#, Html, bootsrap, Css, xUnit. Three tables were created one for Bands, one for Venues, and a join table to store the relationship between both tables

MEAL TRACKER

ABOUT THE PROJECT

An application were a user can log their daily food intake in order to track how many calories they consume in that particular day. There are 3 fileds: Food name/type, calories, and description. A user has the option to view all meals, meals over 500 calories, or meals under 500 calories. Also there is CRUD functionality which allows a user to edit, all three fields if they wish.


LANGUAGES AND TOOLS

Angular 2, Node.js, Sass, Css, bootstrap, Html, and bower were the languages and tools used for this project.

SPEAKERFREAK


ABOUT THE PROJECT
